### Чем я занимаюсь?
- 🛠 Разрабатываю надежные бекенд-решения на **Go** и **Python**.
- 🌐 Проектирую архитектуру и пишу эффективные **REST API**, а сейчас активно погружаюсь в изучение **gRPC**.
- 🧪 Обеспечиваю надежность кода покрытием **Unit-тестами**.
- 🗄 Обладаю глубоким опытом работы с базами данных (**PostgreSQL**, **Redis**, **MongoDB**).
- ⚙️ Уверенно деплою и конфигурирую проекты с помощью **Docker Compose**, **Nginx** и **Ngrok**.
- 🤖 Использую **Python (PyTorch, TensorFlow)** для решения задач машинного обучения.
- ⚡ Увлекаюсь олимпиадным программированием на **C++** (задачи с жесткими Time Limits).
